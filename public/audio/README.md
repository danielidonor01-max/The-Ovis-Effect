# Preloader audio

Drop the intro ambient track here as **`ovis-ambient.mp3`** (this exact name/path:
`public/audio/ovis-ambient.mp3` → served at `/audio/ovis-ambient.mp3`).

The hub preloader's "Enter with sound" button plays it (looped, fades in to ~55%
volume). Until the file exists, "Enter with sound" simply enters silently — no error.

Keep it short and calm (a 15–40s seamless loop is ideal) and reasonably compressed
(≈1–2 MB). Suggested royalty-free sources are in the chat / project notes.
