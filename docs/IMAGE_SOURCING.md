# Image Sourcing Pack — The Ovis Effect

Curated, **free commercial-license** imagery (Pexels / Unsplash / Pixabay — no
attribution required). For each slot: the source link(s), what to pick (Nigerian
context), and the target aspect. Upload in **Sanity Studio** and **set the
hotspot** over the subject so it stays framed on mobile.

General rules: high-res JPG, ≈2× the slot size, subject roughly centred, fill the
**alt text**. Hub cards reuse each brand's **Hero Image** (no separate sourcing).
